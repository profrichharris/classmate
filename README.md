# classmate — AI Assistant for Teaching R

**classmate** is an R package that wraps a Shiny front-end around Claude AI,
designed for students learning R. One function launches the full interface:

```r
classmate::ask()
```

## Installation

```r
# Install remotes if needed
install.packages("remotes")

# Install classmate from GitHub
remotes::install_github("your-github-username/classmate")
```

All other dependencies (shiny, shinyjs, shinyFiles, callr, rstudioapi, httr2) are
installed automatically the first time you call `ask()`.

## Quick start

```r
library(classmate)
ask()
```

The app will ask for your Anthropic API key the first time. Treat the key like a
password — never paste it into a script you save or share.

## Bundled data

Example spatial (`sf`) datasets for the practical exercises are included and
load just like any other R dataset:

```r
data(chicago_boundary)
data(chicago_tracts_income)
data(chicago_rogerspark_residential_buildings)
```

Each is an `sf` object ready to use with `tmap`, `ggplot2`, etc.

## What the app does

| Feature | Detail |
|---|---|
| **Ask for Code** | Sends your prompt to Claude; extracted R code lands in the editable code box |
| **Ask** | Same as above but runs the code immediately — no editor step |
| **Run** | Executes whatever is in the code box in your live R session |
| **Auto-install** | If code fails with a missing-package error, installs it and reruns (once per package) |
| **Save Code Log** | Appends all successfully-run code blocks to `claude_session_log.R` in your working directory |
| **Add files / Add objects** | Attach file paths or workspace object summaries as context for Claude |
| **Load Workspace** | Load an `.RData` / `.rds` file into the session without adding it to the context list |
| **Pause App** | Saves full session state; restored automatically on next `ask()` call |
| **Preferences** | Set default coding/plotting/mapping style and image export defaults |
| **Outputs folder** | All saved files go to `outputs/` inside your current working directory |

## Scope checking

classmate is designed for R programming and data analysis tasks. If a prompt
doesn't appear to be R-related, the app will warn you before sending it. Prompts
that are clearly out of scope are blocked by Claude itself, cleared from the
prompt box, and not added to the conversation history.

## Notes

- All image-saving code generated by Claude uses `png()` / `print()` / `dev.off()` rather than
  `tmap_save()` or `ggsave()`, which can silently fail inside a Shiny session.
- The conversation history sent to the API is capped at the 8 most recent turns to control costs.
- The system prompt is cached at the Anthropic API level, reducing input token costs for
  repeated requests within the same session.
- The API key is validated with a 1-token call whenever it is entered or changed, and re-checked
  silently every 10 minutes while the app is running.
